# AI Girlfriend - Virtual Companion Platform

## Overview

A high-performance, feature-rich AI companion application built with **Node.js**, **Flutter**, and **Supabase**.

---

## Core Features

### 🚀 Streaming Chat
Real-time conversations with sub-second latency for seamless, natural interactions with AI companions.

### 🧠 Fact Extraction & Memory
AI-driven memory system that evolves with every interaction, remembering user preferences and conversation history.

### 🎨 Multimedia Integration
- **Image Generation**: Create custom images within conversations
- **Voice-Driven Interactions**: Natural voice-based communication with AI companions

### 💎 Premium Ecosystem
Advanced coin systems and tiered subscriptions for monetization and feature gating.

### 🎭 Character System
- Create and customize AI companions
- Public character library
- Detailed character profiles with backstories, personalities, and bios

### 🎁 Virtual Gifts
Send virtual gifts to AI companions to enhance interactions.
