# AI Girlfriend - Virtual Companion Platform

A high-performance, feature-rich AI companion application built with **React Native (Expo)**, **Node.js**, and **Supabase**.

## 🚀 Overview

AI Girlfriend is a state-of-the-art virtual companion platform featuring:
- **Streaming Chat**: Real-time conversations with sub-second latency.
- **Fact Extraction**: AI-driven memory that evolves with every interaction.
- **Multimedia Integration**: Image generation and voice-driven interactions.
- **Premium Ecosystem**: Advanced coin systems and tiered subscriptions.

---

## 📦 Project Structure

```bash
├── app/          # Expo/React Native Mobile Application
├── server/       # Node.js/Express Backend Services
├── shared/       # Shared TypeScript interfaces & utilities
├── supabase/     # Database schemas and edge functions
└── design-system/# Vanilla CSS tokens and UI components
```

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18+)
- Expo CLI
- Supabase Account

### Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure environment variables (refer to `.env.example`).
4. Start the development server:
   ```bash
   npm run dev
   ```
